# DelilahRestoProyecto
proyecto acamica

* # paso 1
Clonar el repositorio, ejecutar el comando npm install por consola

* # paso 2
Iniciar los servicios de Xampp, entrar a phpmyadmin y crear la base de datos que se llame Delilah

* # paso 3
Ir al archivo db.js y verificar que la base de datos se llame igual en la conexion en la linea 16.
Ejecutar el comando npm run dev para montar la aplicación, con esto se van a crear las tablas y las relaciones por la librería Sequelize

* # paso 4
Ejecutar el archivo insercionesIniciales en la consola SQL de phpmyadmin para tener los valores iniciales como por ejemplo el estado de los pedidos y metodos de pago

* # paso 5

Listo, ya queda lista para ser testeada
